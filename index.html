Selected Projects

flutter-mcp Dart

一套支持 Model Context Protocol (MCP) 的 Flutter 待办记事本应用。它允许 AI 代理远程读取、检索和编辑日程，实现了代码直接介入客户端 UI 的新型人机协同机制。

cc-switch Rust

跨平台的全功能终端 AI 助手管理器（Fork 自 cc-switch 社区版本）。主要用于简化 Claude Code, Gemini CLI 及各类本地/云端 LLM 代理在终端环境中的无缝切换。

uikit_hacknews Objective-C

基于 iOS 经典 Objective-C UIKit 框架构建的 Hacker News 客户端。项目聚焦于移动端经典的视图层架构设计，实现了新闻条目异步流式解析、缓存与交互展示。

markdown-editor JavaScript

基于 Electron 开发的跨平台极简 Markdown 编辑器。支持实时语法解析、双栏同步滚动预览、快捷键操作以及文件本地管理，专注于沉浸式的轻量化写作体验。
